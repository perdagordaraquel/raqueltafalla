Exercise Reflection

Using internal and external CSS taught me how to separate content from design. Internal CSS, written inside style tags in the head, helps style a single page easily, while external CSS in a separate . css file allows me to apply the same styles to multiple pages. I learned that external CSS makes my website organized and easier to maintain. Practicing these methods showed me that combining inline, internal, and external CSS wisely can make a webpage both functional and visually appealing.
